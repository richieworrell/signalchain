The sizing system is based on the SVG canvas dimensions in the editor, which map to real-world inches via PX_PER_IN = 30 in the layout view.

The formula:

SVG px = real-world inches × 30

So for a standard Boss compact pedal (2.36" × 4.75"):

70 × 142px  (2.36 × 30 = 70.8 → 70, 4.75 × 30 = 142.5 → 142)
